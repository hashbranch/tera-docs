## 1. Get an API key

Tera is currently in private access. Email [hello@tera.gw](mailto:hello@tera.gw?subject=Tera%20API%20access) to request a key. Keys are formatted as `sk-tera-...`.

## 2. Send your first request

<CodeGroup>

```bash curl
curl https://api.tera.gw/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TERA_API_KEY" \
  -d '{
    "model": "Qwen/Qwen2.5-7B-Instruct",
    "messages": [
      {"role": "user", "content": "Say hello in one short sentence."}
    ],
    "max_tokens": 64
  }'
```

```python python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.tera.gw/v1",
    api_key="sk-tera-...",
)

resp = client.chat.completions.create(
    model="Qwen/Qwen2.5-7B-Instruct",
    messages=[{"role": "user", "content": "Say hello in one short sentence."}],
    max_tokens=64,
)
print(resp.choices[0].message.content)
```

```javascript node
import OpenAI from "openai";

const client = new OpenAI({
  baseURL: "https://api.tera.gw/v1",
  apiKey: process.env.TERA_API_KEY,
});

const resp = await client.chat.completions.create({
  model: "Qwen/Qwen2.5-7B-Instruct",
  messages: [{ role: "user", content: "Say hello in one short sentence." }],
  max_tokens: 64,
});

console.log(resp.choices[0].message.content);
```

</CodeGroup>

## 3. Stream the response

Add `"stream": true` to get token-by-token output over Server-Sent Events. See [Streaming](/concepts/streaming) for the full event format.

```bash
curl https://api.tera.gw/v1/chat/completions \
  -H "Authorization: Bearer $TERA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "Qwen/Qwen2.5-7B-Instruct",
    "messages": [{"role": "user", "content": "Count to five."}],
    "stream": true
  }'
```
